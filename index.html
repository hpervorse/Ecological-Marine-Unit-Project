Marine Sanctuary Project

A project exploring Flower Garden Banks National Marine Sanctuary with goals to increase sanctuary boundaries.

See Maps

About The Sanctuary

Working with Flower Garden Banks National Marine Sanctuary (FGNMS) this project performed seafloor analysis of the seafloor and water column to identify areas to protect.

The sanctuary is one of 14 federally designated underwater areas protected by NOAA's Office of National Marine Sanctuaries. The Draft Environmental Impact Statement has been submitted and this project aims to identify the ecological significance of the proposal areas using various analysis methods.

Project Details

This project uses data from USGS and the sanctuary to perform seafloor analysis. Idenfitying groups of similar seafloor characteristics (slope, benthic position index, terrain ruggedness) maps have been made to show areas that have the charcteristics the sanctuary is aiming to protect.

Using ArcGIS Pro geoprocessing tools and various functionality in the software this project created maps and outputs useful to the sanctuary in their expansion goals.
